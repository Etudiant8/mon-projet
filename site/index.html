Livraison automatisée d'un site Nginx via GitHub Actions

Cette page est servie par Nginx dans une image Docker construite, testée, publiée sur GHCR et promue automatiquement, sans reconstruction, jusqu'à un environnement de production simulée.

Chaîne CI/CD

Commit & push vers GitHub Versionnement du code source, déclenchement automatique des workflows.
01-ci.yml — Build & test HTTP Construction de l'image Docker et test HTTP automatisé sur le port 8080.
02-publish-ghcr.yml — Publication GHCR Image taguée (sha- + latest) et publiée dans GitHub Container Registry.
03-promote.yml — Validation recette Test de l'image existante dans l'environnement GitHub « recette », sans rebuild.
Promotion production-simulée Promotion manuelle (workflow_dispatch) du même artefact — aucune reconstruction.
